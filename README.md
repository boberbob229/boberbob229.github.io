# Tinfoil Hat Linux (Archive)

This repo is a preservation copy of Tinfoil Hat Linux, an old floppy-based security distro from the early 2000s.

It includes the original disk image and a mirror of the website as it existed when retrieved. The project appears long discontinued.

This is kept here for historical/reference purposes only.

## Notes

- This system is outdated and not secure.
- Do not use it on real hardware or networks.
- If you want to try it, run it in a VM.

Checksums are included so the image can be verified.

If you're the original author and want anything changed, feel free to reach out.
